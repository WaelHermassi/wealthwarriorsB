📊 Stratégie 1: Kelly Criterion Binaire
Formule corrigée pour options binaires

La formule Kelly classique ne tient pas compte du payout. Version binaire exacte :

f = (WR × payout − (1−WR)) ÷ payout
À 55% WR + 92% payout → f ≈ 6% → Half-Kelly = 3%
⚠️ Si f ≤ 0 : edge négatif → ne pas trader à ce WR/payout
Half-Kelly = Kelly ÷ 2. Réduit la volatilité tout en maintenant 75% de la croissance optimale.
🚀 Accélération → Mise : 2.500% = $108.18 · min $1 · cap 2.5%
📈 Stratégie 2: Fixed Fractional (Payout-Aware)
Risque fixe avec breakeven réel affiché

Vous risquez un pourcentage fixe du capital. Le breakeven réel dépend du payout, pas du ratio 1:1.

📌 Breakeven réel = pertes ÷ (WR × payout)
À 55% WR + 92% payout → breakeven WR = 52.1% ✅
Vos 55% WR sont au-dessus → edge positif garanti
Edge net par trade = 55%×0.92 − 45% = +0.6%
🚀 Accélération → Mise : 2.500% = $108.18 · min $1 · cap 5%
🎯 Stratégie 3: Martingale Payout-Aware
Multiplicateur exact + protection plafond séries

Récupère les pertes avec le multiplicateur minimal qui garantit un profit net après un win, payout inclus.

📌 Multiplicateur exact :
multi = (1 + payout) ÷ payout
À 92% payout → multi = 1.92 ÷ 0.92 = ×2.087
⛔ Plafond séries : au-delà de N losses → reset automatique
Après ce nombre de losses → reset à la mise de base (protection capital)
🚀 Accélération → Base : $7.86 · ×2.087 · Max 6L · Worst: $649.09
🚀 Stratégie 3b: Anti-Martingale (Paroli)

Augmente la mise après chaque WIN — profite des streaks gagnants. Reset automatique après 3 wins consécutifs (banking des gains).

Multiplicateur plafonné à 1.75× pour limiter l'exposition. Reset automatique après 3W consécutifs.

⚖️ Stratégie 4: D'Alembert Asymétrique (Payout Compensated)
🎯 Solution mathématique pour le payout 92%

Le D'Alembert classique échoue car +1u gain ≠ -1u perte avec un payout de 92%. Cette version corrige l'asymétrie.

⚡ Incrément Asymétrique :
📈 Après un GAIN → Mise −1 unité
📉 Après une PERTE → Mise +1u ÷ 0.92 ≈ +1.087 unité
✅ Résultat : 1 Win annule exactement 1 Loss, frais courtier inclus.
Avec 5W + 5L dans n'importe quel ordre → toujours en positif.
Exemple (base 10$ / step 1%):
Perte → mise passe de 10$ à 11.09$ (+1.087u)
Ensuite Win → 11.09 × 0.92 = +10.20$ → couvre la perte de 10$ ✅
L'incrément réel après perte sera automatiquement calculé : unité ÷ payout (ex: 0.5% ÷ 0.92 ≈ 0.543%)
🚀 Accélération → Base : $103.85 · Step $34.62 (+0.870%/perte) ✅ Actif
